A simple tweak that enables the native touch for macOS 27 on Sidecar for older iPadOS versions.

See [this video](https://www.reddit.com/r/MacOSBeta/comments/1u15hku/macos_27_db1_macos_has_full_touchscreen_support/) for a demo of the new feature, which officially requires iPadOS 27 and macOS 27.

Tested and verified working: 
- iPad Air M1 on iOS 16.1.1
- iPad 7th Gen on iOS 18
